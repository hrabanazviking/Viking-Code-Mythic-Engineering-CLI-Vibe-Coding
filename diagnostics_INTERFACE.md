# diagnostics.py — INTERFACE.md

## Class: `DiagnosticSystem`

Comprehensive diagnostic and troubleshooting tools.

### `run_all_checks()`
Run all diagnostic checks.

### `check_python()`
Check Python version and environment.

### `check_dependencies()`
Check required dependencies.

### `check_config()`
Check configuration file.

### `check_directories()`
Check required directories.

### `check_data_files()`
Check essential data files.

### `check_characters()`
Check character files.

### `check_sessions()`
Check session files.

### `check_memory_system()`
Check memory system files.

### `check_auto_generated()`
Check auto-generated content.

### `print_summary()`
Print summary of issues and warnings.

### `save_report(filepath)`
Save diagnostic report to file.

### `auto_fix()`
Attempt to automatically fix issues.

## Module Functions

### `main()`
Run diagnostics.
