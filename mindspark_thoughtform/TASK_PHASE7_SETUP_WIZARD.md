**Detection logic:**
- Check `ollama` in PATH, try `GET http://localhost:11434/api/tags`
- Check LM Studio at `http://localhost:1234/v1/models`
- Check llama-cpp-python importable
- Check CUDA: `nvidia-smi`, ROCm: `rocm-smi`, Metal: `platform.system() == Darwin`

**Config written to `configs/user_config.yaml`:**
```yaml
backend: ollama
model: llama3.2:3b
ollama_url: http://localhost:11434
lmstudio_url: http://localhost:1234
hf_token: ""
openai_api_key: ""
openai_base_url: ""
profile: auto
knowledge_db_built: true
```

**Key behaviours:**
- Re-runnable — detects existing config and asks "reconfigure? [y/N]"
- All prompts have sensible defaults (Enter = skip/default)
- Never stores keys in plaintext if user prefers env vars — offers both
- Validates chosen backend with a test call before declaring success
- If test fails, offers to try another backend

### `src/thoughtforge/inference/unified_backend.py`

```python
class GenerationRequest:
    prompt: str
    temperature: float = 0.7
    max_tokens: int = 512
    stop: list[str] = []
    system_prompt: str = ""
    messages: list[dict] = []   # for chat-mode backends

class GenerationResponse:
    text: str
    tokens_generated: int
    finish_reason: str   # "stop" | "length" | "error"
    backend_used: str
    latency_ms: float

class UnifiedBackend(ABC):
    def generate(self, request: GenerationRequest) -> GenerationResponse: ...
    def health_check(self) -> bool: ...
    def backend_name(self) -> str: ...

def load_backend_from_config(config_path=None) -> UnifiedBackend:
    """Read user_config.yaml and return the appropriate backend instance."""
```

**Concrete backends:**
- `TurboQuantBackend` — wraps existing llama-cpp TurboQuantEngine
- `OllamaBackend` — HTTP to `ollama_url/api/generate` or `/api/chat`
- `LMStudioBackend` — OpenAI-compatible `/v1/chat/completions`
- `HuggingFaceBackend` — `huggingface_hub.InferenceClient`
- `OpenAICompatibleBackend` — generic OpenAI-compatible (vLLM, text-gen-webui, etc.)

**All backends implement:**
- Streaming support (optional, yields tokens)
- Timeout handling with clear error messages
- Retry on transient failure (3 attempts, exponential backoff)

### `src/thoughtforge/inference/model_browser.py`

```python
class ModelEntry:
    model_id: str       # e.g. "microsoft/Phi-3-mini-4k-instruct-gguf"
    filename: str       # e.g. "Phi-3-mini-4k-instruct-q4.gguf"
    size_gb: float
    tier: str           # phone | pi | cpu | gpu | server
    description: str

class ModelBrowser:
    CURATED_MODELS: list[ModelEntry]   # ~20 well-tested models
    
    def search_hf(self, query: str) -> list[ModelEntry]: ...
    def download(self, entry: ModelEntry, dest: Path, progress=True) -> Path: ...
    def list_ollama_library(self) -> list[str]: ...
```

Curated list covers all hardware tiers. Uses `huggingface_hub.hf_hub_download` with
progress bar via `tqdm`. Validates SHA256 after download.

### `src/thoughtforge/cognition/chat_history.py`

```python
@dataclass
class ChatMessage:
    role: str       # "user" | "assistant" | "system"
    content: str
    turn_id: str
    timestamp: str

class ChatHistory:
    max_turns: int = 20
    messages: list[ChatMessage]
    
    def add_user(self, text: str) -> None: ...
    def add_assistant(self, text: str, turn_id: str) -> None: ...
    def to_prompt_context(self, max_chars: int = 2000) -> str: ...
    def to_messages(self) -> list[dict]: ...   # OpenAI format for API backends
    def trim_to_budget(self, token_budget: int) -> None: ...
    def save(self, path: Path) -> None: ...
    def load(self, path: Path) -> None: ...
```

`ThoughtForgeCore.think()` gains optional `history: ChatHistory | None = None` param.
History context injected into scaffold + prompt builder.

### `run_thoughtforge.py` — Chat Mode

New flags:
```
--chat              Enter persistent chat mode (default when no query given)
--history FILE      Load/save chat history from file
--no-history        Disable history persistence
--system PROMPT     Override system prompt for this session
```

Chat mode loop:
```
ThoughtForge Chat — llama3.2:3b via Ollama
Type 'exit' to quit, '/clear' to reset history, '/save' to save, '/help' for commands.

You > hello, how are you?
Forge > Well met. The forge runs warm and steady — ready to reason, recall, or ramble
        on whatever you bring. What calls you here today?

You > what is yggdrasil?
Forge > Yggdrasil is the immense ash tree at the center of Norse cosmology...
        [Citations: Q42240 | Confidence: 0.841 | PASS]

You > /save
✓ History saved to chat_2026-03-31.json
```

In-chat commands: `/clear`, `/save`, `/load FILE`, `/stats`, `/help`, `/model`, `/backend`
